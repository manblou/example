
Config steps for bank net design

1. basic setttings to all devices pluss ssh on the routers and 13 swithces
2. VLANs assignment plus all access and trunk ports
3. Switchport security to all 12 switches
4. Subnetting and IP addressing
5. OSPF on the routers and 13 swiitches
6. Static IP address to server room devices
7. DHCP SERVER device configuration
8. Inter-VLAN routing on the 13 switches
9. Wirless network configuration
10. verifying and testing configuration



________access layer switch_______


enable
conf t
hostname F04-svr-sw
banner motd #This is floor-04 server room swith#
line console 0
password banknet
login
exit
line vty 0 15
password banknet
login
exit
no ip domain-lookup
enable password banknet
service password-encryption


int range fa0/1-2
switchport mode trunk
exit

int range fa0/3-24
switchport mode access
switchport access vlan 120
switchport port-security
switchport port-security maximum 2
switchport port-security mac-address sticky
switchport port-security violation shutdown
exit
do write
